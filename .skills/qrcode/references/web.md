# QR Code — Web / H5

Use the scaffold's built-in `QRCodeDataUrl` component for pure frontend QR code generation.

## Usage

```typescript
import { QRCodeDataUrl } from '@/components/ui/qrcodedataurl'

<QRCodeDataUrl data="https://example.com/path" />
```

The component generates a QR code image as a data URL — no backend or Edge Function needed.
