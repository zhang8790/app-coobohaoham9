# Anti-Slop 设计约束清单

> 机制：脚本只提取每条**加粗标题**注入搜索结果；破折号说明仅供维护。

## 5 条设计硬约束（写 DESIGN.md 时必须遵守）

1. **禁大块纯色铺底，含 Hero/Banner/上传/操作/输入区；Primary/Accent 仅小面积焦点** — 大面积区域用中性底、纹理、图像或分层
2. **禁用圆角卡片/细描边/通用投影/左色条/渐变背景充当核心视觉签名** — 这些只能做辅助
3. **核心视觉签名必须来自字体排版、纹理材质、图像处理、局部光影或图标语言** — 签名要可记忆且可编码
4. **禁 Emoji 图标、CSS 伪装 Logo、系统默认展示字体** — 使用正式图标/字体资源
5. **禁风格混搭；非灰阶色彩角色 ≤ 4；禁循环装饰动效** — 选一种方向彻底执行
