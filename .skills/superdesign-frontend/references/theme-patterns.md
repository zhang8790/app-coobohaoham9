# 主题模式参考

本文档提供 3 种预设主题模式的完整 CSS 变量定义，Agent 可根据用户需求直接选用或在此基础上调整。

---

## 1. 现代深色模式（Vercel / Linear 风格）

干净、专业、高对比度的现代深色风格。适合 SaaS 产品、开发者工具、仪表盘。

### 适用场景

- SaaS 产品官网/仪表盘
- 开发者工具界面
- 需要长时间使用的后台系统
- 追求简约专业感的项目

### CSS 变量

```css
:root {
  /* 背景与文字 */
  --背景色: oklch(0.145 0 0);
  --前景色: oklch(0.985 0 0);
  --主色: oklch(0.985 0 0);
  --主色前景: oklch(0.145 0 0);

  /* 辅助色 */
  --次色: oklch(0.269 0 0);
  --次色前景: oklch(0.985 0 0);
  --静音色: oklch(0.269 0 0);
  --静音前景: oklch(0.708 0 0);

  /* 边框与装饰 */
  --边框色: oklch(0.269 0 0);
  --圆角: 0.625rem;

  /* 字体 */
  --无衬线字体: Inter, system-ui, -apple-system, "PingFang SC", "Microsoft YaHei", sans-serif;
  --等宽字体: JetBrains Mono, "Fira Code", "Source Code Pro", monospace;
}
```

### 设计要点

- 背景使用极深的灰色（接近纯黑），前景使用纯白或极浅灰
- 次级元素使用稍浅的灰色区分层级
- 边框使用半透明或低对比度线条，避免生硬分割
- 圆角适中（0.5-0.75rem），保持现代感
- 字体推荐 Inter，中文回退至 PingFang SC / Microsoft YaHei

## 2. 新粗野主义（90 年代网页复兴）

大胆、醒目、反常规的设计风格。高饱和度色彩、粗黑边框、零圆角。适合创意项目、艺术网站、年轻化品牌。

### 适用场景

- 创意机构官网
- 艺术/设计类个人作品集
- 年轻化、潮流感品牌
- 需要强烈视觉冲击力的营销活动页

### CSS 变量

```css
:root {
  /* 高饱和度色彩 */
  --背景色: oklch(1 0 0);
  --前景色: oklch(0 0 0);
  --主色: oklch(0.649 0.237 26.97);      /* 鲜红 */
  --次色: oklch(0.968 0.211 109.77);     /* 明黄 */
  --强调色: oklch(0.564 0.241 260.82);   /* 亮蓝 */

  /* 粗野风格特征 */
  --边框色: oklch(0 0 0);
  --边框宽度: 3px;
  --圆角: 0px;                           /* 零圆角！ */
  --阴影: 4px 4px 0px 0px hsl(0 0% 0%);  /* 硬阴影 */

  /* 字体 */
  --无衬线字体: "DM Sans", "Arial Black", "Heiti SC", sans-serif;
  --等宽字体: "Space Mono", "Courier New", monospace;
}
```

### 设计要点

- **零圆角**：所有元素保持直角，强化硬朗感
- **粗黑边框**：3px 及以上宽度的纯黑边框
- **硬阴影**：不使用模糊阴影，使用纯色偏移阴影（如 `4px 4px 0 0 black`）
- **高饱和度**：使用鲜艳的红、黄、蓝，避免 pastel 色调
- **大胆排版**：标题字号夸张，字重偏粗

## 3. 玻璃拟态（Glassmorphism）

半透明毛玻璃效果，轻盈、通透、富有层次感。适合展示类产品、天气应用、音乐播放器、现代创意网站。

### 适用场景

- 展示类/作品集网站
- 天气/音乐等视觉驱动型应用
- 需要层次感和通透感的界面
- 与视频或动态背景结合的设计

### CSS 实现

```css
/* 玻璃效果基础类 */
.玻璃 {
  background: rgba(255, 255, 255, 0.1);
  backdrop-filter: blur(10px);
  -webkit-backdrop-filter: blur(10px);  /* Safari 兼容 */
  border: 1px solid rgba(255, 255, 255, 0.2);
  border-radius: 1rem;
  box-shadow: 0 4px 30px rgba(0, 0, 0, 0.1);
}

/* 深色背景上的玻璃效果 */
.玻璃-深色 {
  background: rgba(0, 0, 0, 0.2);
  backdrop-filter: blur(16px);
  -webkit-backdrop-filter: blur(16px);
  border: 1px solid rgba(255, 255, 255, 0.1);
  border-radius: 1rem;
  box-shadow: 0 8px 32px rgba(0, 0, 0, 0.3);
}
```

### 设计要点

- **半透明背景**：白色或黑色透明度 10%-25%
- **背景模糊**：`backdrop-filter: blur(10-20px)` 是核心
- **细边框**：半透明白色边框（1px），增强玻璃边缘感
- **微妙阴影**：柔和的弥散阴影，增加悬浮感
- **背景要求**：玻璃效果需要背后有丰富色彩/图案才能体现效果，纯白墙无效
- **性能注意**：`backdrop-filter` 在低端设备和某些浏览器上性能较差，需测试

## 色彩选择建议

### 避免使用的色彩

| 色彩 | 原因 |
|------|------|
| `#007bff`（Bootstrap 蓝） | 过于常见，显得过时、缺乏品牌感 |
| 纯黑 `#000000` | 在屏幕上过于刺眼，建议使用 `#0a0a0a` 或 `oklch(0.145 0 0)` |
| 纯白 `#ffffff` | 在深色主题中可能过亮，建议使用 `oklch(0.985 0 0)` |

### 推荐做法

- 使用 `oklch()` 定义色彩，人眼感知更均匀
- 定义语义化变量：`--主色`、`--次色`、`--静音色`、`--边框色` 等
- 同时设计浅色和深色模式，使用 `@media (prefers-color-scheme: dark)` 或 CSS 变量切换
