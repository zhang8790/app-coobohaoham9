# 章节循环

每章运行此循环以保护质量和写作动力。

## 1. 定义章节契约

- **成果**：读者在本章结束后能做什么、理解什么或感受到什么。
- **输入**：来自前面章节的先决上下文。
- **边界**：哪些内容属于本章，哪些属于后面的章节。

## 2. 单向起草

- 从头到尾起草，不做微观编辑。
- 在不确定的地方用简单占位符标记，如 `[核实来源]`。
- 初稿流中避免完美主义修改。

## 3. 验证连贯性

定稿前快速检查：

- 开头是否匹配承诺的成果？
- 每一节都有存在的价值吗？
- 是否有一条清晰的主线？
- 结尾是否自然地过渡到下一章？

## 4. 记录决策

在记忆中记录：

- 范围变更。
- 声音或语气调整。
- 可能影响未来章节的新风险。
