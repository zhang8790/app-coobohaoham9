# 记忆模板

使用此结构创建 `~/book-writing/memory.md`：

```markdown
# 书籍写作项目记忆

## 状态
status: ongoing
version: 1.0.0
last: YYYY-MM-DD
integration: pending | done | declined

## 项目快照
- 工作标题：
- 书籍类型：
- 目标读者：
- 核心承诺：
- 当前阶段：

## 声音档案
- 语气：
- 视角和时态：
- 句式节奏：
- 避免使用的词汇或模式：

## 书籍地图
| 章节 | 读者成果 | 状态 | 备注 |
|------|----------|------|------|
| 1 | | planned | |

## 活跃风险
- 风险：
  影响：
  缓解措施：

## 下一步行动
- [ ]
- [ ]

## 笔记
- 关键决策和理由。
```

## 状态值

| 值 | 含义 | 行为 |
|----|------|------|
| `ongoing` | 上下文仍在演变 | 在正常流程中持续学习 |
| `complete` | 基线已稳固 | 聚焦执行和交付 |
| `paused` | 用户请求暂停 | 保持上下文，避免设置提示 |
| `never_ask` | 用户永久退出 | 不再询问设置上下文 |

## 记忆卫生

- 保持条目简洁且以行动为导向。
- 优先记录稳定模式，而非一次性观察。
- 将过时的方向变更移到 `archive/` 笔记中，不要留在活跃记忆里。
