# 高端会员经济 · 设计方案（情绪资产体系）

> 目标：把现有「5+ 种币值并存、命名错位、段位与情绪脱节」收敛为一套**高端、合规、可收藏**的会员资产体系。
> 原则：**合理分配**（用户只看见 3 件套）+ **更高端玩法**（身份感、收藏感、仪式感），全程不碰资金合规红线。

---

## 〇、品牌叙事（一句话定调）

每一次消费 = 一次「情绪确权」：把一次购物升华为一份**被接住的情绪凭证**。

- **金豆** = 流通币（你花出去的）
- **情绪豆** = 情感资产（平台内情绪玩法，喂养 / 兑换专属体验）
- **贡献值** = 成长轨迹（驱动段位晋升）
- **段位** = 身份铭牌（六阶武侠身份，对应专属权益）
- **徽章** = 情绪图鉴（按情绪维度收藏，稀有度分级）

> 命名去金融化：一律不出现「分红 / 股权 / 返现 / 投资」。权益用「专属服务 / 优先体验 / 共创权 / 荣誉标识」表述。

## 一、三件套资产（合理分配）

| 资产 | 来源 | 用途 | 合规边界 |
|---|---|---|---|
| **金豆** `gold_beans` | 消费获得（原 `points` 1:1 并入）+ 活动 | 订单 1:1 抵扣 | 不可提现、不可兑现金、不可二级转让 |
| **情绪豆** `tb_balance` | 确权发放（消费即确权） | 喂养情绪、兑换专属食养·情绪体验 | 仅平台内玩法，不可提现/兑现金/转让 |
| **贡献值** `cv_total` | 确权发放 + 消费累计 | 段位晋升主依据 | 仅身份成长，无现金属性 |

**关键决策**：删除 `points`，历史余额 1:1 合并进 `gold_beans`（用户无感）；`points` 与 `gold_beans` 本就是同质 1:1 抵扣币，抵扣链路实际只用 `gold_beans`。情绪豆**不**并入——它有独立玩法，不能当通用抵扣币。

用户端只露出这 3 个 + 段位 + 徽章，彻底消除「积分 + 消费积分 + 贡献值(还被误标成情绪豆) + 情绪豆」的混乱。

## 二、段位六阶（高端身份 · 合规权益）

段位由「个人累计消费」决定基础门槛（与佣金比例所用的消费段位**完全一致**，避免两套阈值触碰资金合规）；高段位（长老 / 掌门）叠加「徽章收集度」软门槛（长老 ≥ 4 枚、掌门 ≥ 8 枚且含史诗/传说）。团队 / 拓新**仅作推广佣金系数，不进入段位本身**（避免「等级靠拉人」观感，合规）。
> 说明：贡献值(cv_total) 不进入段位门槛，仅驱动「年度成长回馈」份额（情绪豆）。身份段位(syncMemberRank) 与佣金段位(消费) 解耦但同源，规则可解释、可审计。

| 段位 | 身份感 | 高端权益（合规） | 晋升软参考 |
|---|---|---|---|
| 江湖散修 | 入门 | 情绪确权基础礼包 | — |
| 外门弟子 | 入门+ | 月度「情绪顾问」轻咨询 ×1 · 生日情绪礼 | — |
| 内门弟子 | 进阶 | 新品优先体验 · 专属食养配方卡 | — |
| 核心弟子 | 中坚 | 1v1 情绪管家（季度）×1 · 线下主题沙龙邀约 | 累计确权 ≥ N |
| 长老 | 高阶 | 食养·情绪 共创权（命名/配方投票）· 私享品鉴 | 徽章收集度 ≥ N |
| 掌门 | 荣誉 | 荣誉身份铭牌 · 年度成长盛典邀约 · 掌门专属客服 | 徽章稀有度达标 |

> 高段位（核心弟子↑）晋升时，**参考「累计确权次数 / 徽章收集度」作为软门槛（不硬卡）**——既关联徽章，又不让"没徽章就卡死升级"产生负面体验。

---

## 三、徽章图鉴（收藏玩法 · 高端感核心）

- **稀有度分级**：`common → rare → epic → legend`；`legend` 带流光动效。
- **图鉴墙**：按 6 大情绪维度（drained_low / lonely_still / expressive_high / peaceful_zen / nostalgic_soft / eager_forward）分类收集，集齐一套解锁**限定徽章**或专属权益。
- **反向价值**：稀有徽章（epic / legend）解锁**专属身份标识**，让徽章拥有超越段位的稀缺价值。
- 与段位解耦但呼应：徽章是「情绪成就装饰」+ 高段位软门槛，不互相硬卡。

---

## 四、段位晋升仪式（动效 · premium 触感）

晋升瞬间：全屏流光 → 段位铭牌揭晓 → 新权益卡片展开（glass morphism + 磁性按钮）。让"升级"从一行文案变成**有记忆点的仪式**，这是"更高端玩法"的情绪落点。

## 五、统一资产视图（UI 收敛）

「我的」一屏资产区重做为 **premium 三件套卡片**：

```
[ 金豆  1,280 ]   [ 情绪豆  320 ]   [ 贡献值  1,640 ]
                              └─ 段位进度条：内门弟子 → 核心弟子（还差 360 贡献值）
[ 徽章图鉴 入口 › ]                         [ 我的段位 / 权益 › ]
```

- 金豆：大数字卡，琥珀金渐变。
- 情绪豆：紫调卡，点进去是喂养 / 兑换。
- 贡献值：青调卡，下方显示段位进度条（当前段位 → 下一阶 + 差额）。
- 徽章：图鉴入口，展示已收集数 / 总数。

## 六、合规护栏（贯穿全案）

1. **段位不挂钩拉人头 / 多级**（已限两级：好友 / 粉丝）。
2. **权益不承诺现金回报 / 保本**；不出现「分红 / 股权 / 返现」表述。
3. **命名去金融化**：贡献值 / 成长值 优于「积分」；顶层权益用「专属顾问 / 成长礼 / 共创权」。
4. **计算透明可查**：rank-rules 公示「累计消费 + 贡献值决定」，不暗箱。
5. **币种协议明示**：金豆 / 情绪豆在协议与余额页标注「仅平台内使用，不可提现 / 兑现金 / 二级转让」，与可提现佣金（`commission_balance`，真实服务费、代扣个税）严格隔离。
6. **医疗宣称零容忍**：食养功效措辞 = 传统食养参考，禁「治疗 / 治愈」等。
