- 正确做法：`ALTER TABLE xxx DISABLE ROW LEVEL SECURITY`（关闭）
- 关闭 RLS 后重新插入全部测试数据，验证 API 可正常读写
- **教训**：Supabase 新项目默认开启 RLS，测试阶段需先 `DISABLE`
