### 问题 6：SQL 脚本语法错误 ✅ 已修复
- **错误**：`CREATE POLICY IF NOT EXISTS "Allow all" ON public.commissions`
- **根因**：PostgreSQL 不支持 `CREATE POLICY IF NOT EXISTS` 语法
- **修复**：改为 `DROP POLICY IF EXISTS` + `CREATE POLICY`
- **文件**：`supabase/COMPLETE_INIT.SQL`
