## 红包功能说明

### 用户领取入口
- 首页活动banner → 点击进入 `pages/campaign-claim/index?campaignId=xxx`
- 需要先阅读并同意活动规则
- 领取后自动建立锁客关系（180天有效期）

### 商家发放入口
- 商家中心 → 快捷操作 → "发放红包"按钮
- 或：商家中心 → 快捷操作 → "管理活动"按钮
- 可以创建现金红包或实物礼品活动
- 设置红包金额、发放总数、每日限领、佣金比例等

### 数据库函数
- `claim_campaign`：领取营销活动奖励（包含锁客逻辑）
- 位于：`supabase/migrations/20260705_update_claim_campaign_with_lock.sql`
