## 二、四类高危词扫描结果

| 维度 | 结果 | 说明 |
|------|------|------|
| ① 传销红线（分销/下线/锁定/层级/团队计酬） | 用户可见 **0 违规** | 残留仅后端算法词「裂变附加分」+ 反传销警示语（正确保留） |
| ② 广告法极限词（最/第一/顶级/国家级/100%…） | **无真实违规** | 命中全为 CSS `100%`、`最高段位`(=最高等级)、`第一屏`(=首屏)，均合法 |
| ③ 医疗功效宣称（治疗/治愈/抗炎/降血压…） | **合规** | 食养字典 `shiyang-dictionary.ts` 带「传统食养参考 + 不替代医嘱」声明 |
| ④ 虚假收益/赌博诱导（稳赚/躺赚/暴富/赌…） | **无违规** | 仅防护性红词库与 RiskWarning 警示语（属防护，非违规） |

### 关键判断：红线在「人」，不在「级数」
- 所谓"三级以内合法"是误读；真正红线是 **拉人头 / 入门费 / 团队计酬**。
- 本产品为**二级推广 + 真实商品零售 + 段位比例由个人消费决定**，且全站明示"不按团队计酬、无入门费、仅两级"——符合合规框架。
- 反传销警示语（distribution-agreement / commission-rules / my-promotion）**必须保留**，是降低法律风险的关键证据。

## 三、本轮「二级推广」专项修复清单

### 1. 模型口径纠偏（原写"仅直推"，与后端 L1+L2 矛盾 → 改回"二级推广"）
| 文件 | 修正 |
|------|------|
| my-promotion/index.tsx | 推广说明改为「直推奖励 + 间推奖励 + 仅二级…」；佣金概览恢复「间推奖励(L2%)」；头部「直推团队」→「推荐团队」 |
| commission-rules/index.tsx | 奖励来源改为「一级推荐好友(直推)/二级推荐好友(间推)」；比例改为「直推 15~28% + 间推平台统一」；明确「只有二级、不发展第三级」 |
| distribution-agreement/index.tsx | 奖励规则改为「仅限两级（直推+间推）」；禁止条款「不得发展**三级及以上**多层级」；法律风险告知「推广为二级结构，仅限二级」 |

### 2. 全站违规词清理（用户可见文案 + 关键注解）
| 词 | 替换 | 涉及文件（节选） |
|----|------|------|
| 分销 | 推广 | user(分销团队→推广团队)、help(推广分销→推广奖励)、merchant-settings(让利（分销）→推广)、types/api(分销佣金→推广佣金)、RiskWarning 注解 |
| 下线 | 推荐关系/推荐 | merchant-center(注册即成为您的**推荐关系**)、my-referrals 注解、types(新增下线→新增推荐客户)、commission-helpers |
| 锁客/锁定(人) | 归属/绑定 | merchant-members(本店**客户**数/客户列表/归属客户名单)、employee(查看**归属**客户)、index(绑定专属门店优惠)、store-home/merchant-center/campaign-claim/share.ts/article-detail 注解 |
| 永久锁定 | 永久删除 | 原侠客推广中心「永久锁定为一经下线」→「成为你的推荐好友」；rank-rules「永久享受」→「须保持活跃」 |

### 3. 命名统一（解决"同名不同字"）
- 模型名统一：**二级推广**（原混用"两级推广"，已全改"二级"）
- 关系称谓统一：**推荐关系 / 归属客户**（不再用"下线/锁客"）
- 奖励/层级称谓：基准口径为「一级/二级(指人)」「直推奖励/间推奖励」，详见下方"社交化口径"更新。

### 3.1 社交化术语更新（2026-07-10 最新口径，覆盖全站）
- 用户指令："推广团队都修改成我的好友，一级是我的好友，二级是我的粉丝"。
- 用户可见映射：`推广团队`→`我的好友`；一级(指人)→`我的好友`、二级(指人)→`我的粉丝`；`直推`→`我的好友`、`间推`→`我的粉丝`；my-referrals `直接推荐/间接推荐`→`我的好友/我的粉丝`。
- 合规铁律不动：描述"两级模型"本身的 `二级推广/二级结构/只有二级/仅二级/仅限两级/第三级/多级分润/多层返利` 一字未动，仅括号内说明 `(直推+间推)`→`(我的好友+我的粉丝)`。
- 含义：一级=我直接邀请的人=我的好友；二级=我的好友再邀请的人=我的粉丝。两层封顶，不发展多级。

### ✅ 协议页风格统一（2026-07-10 已修复）
