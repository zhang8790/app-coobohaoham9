# 需求文档

## 1. 应用概述

### 1.1 应用名称
来店有喜 V3 终极版

### 1.2 应用描述
本地生活服务类电商微信小程序,以武侠/国潮风格为设计基调,通过情绪输入引擎匹配用户需求,提供商品购买、本地服务预订、UGC内容发布、分销推广等功能。

## 2. 用户与使用场景

### 2.1 目标用户
- 本地生活服务消费者
- 寻求情绪化购物体验的用户
- 希望通过分销获取收益的推广者
- 本地商家
- 平台超级管理员

### 2.2 核心使用场景
- 用户根据当前情绪状态浏览匹配的商品和服务
- 用户购买商品或预订本地服务
- 用户发布UGC内容分享体验
- 分销推广者邀请新用户获取佣金
- 商家入驻平台开展业务
- 超级管理员审核商家申请、商品、提现、UGC内容

### 3.2 TabBar主页面功能说明

#### 3.2.1 首页(index)

**核心功能**:情绪化商品推荐与内容浏览

**功能模块**:
- 情绪输入引擎:用户输入当前情绪或心情描述
- IP伴侣气泡:展示情绪化提示语,引导用户互动
- 快捷情绪词:提供常用情绪词快速选择
- 状态选择栏:用户选择当前时间状态(早晨/午后/傍晚/夜晚)或日期状态(工作日/周末/假期)
- 公告滚动:展示平台公告信息
- 场景卡片:展示4个场景入口(治愈空间/用餐时光/购物时刻/学习空间),点击进入对应场景商品列表
- 双列瀑布流Feed:根据情绪匹配和附近优先原则展示商品卡片,支持上拉加载更多
- 悬浮按钮:提供扫码和创作两个快捷入口

#### 3.2.2 探索页(explore)

**核心功能**:商品分类浏览与购买

**功能模块**:
- 搜索栏:点击进入搜索页
- 扫码入口:扫描商品二维码
- 购物车入口:显示购物车商品数量,点击进入购物车页面
- 左侧分类导航:展示商品分类(全部/图书/美食/饮品/零食/日用/礼品),点击切换分类
- 右侧商品网格:展示当前分类下的商品列表,每个商品卡片包含商品图片、名称、价格、加购按钮
- 加购按钮:点击将商品加入购物车
- 上拉加载更多:加载更多商品

#### 3.2.3 犒赏铺(reward-shop)

**核心功能**:本地服务商家浏览与预订

**功能模块**:
- 搜索栏:点击进入搜索页
- 购物车入口:显示购物车商品数量,点击进入购物车页面
- 左侧服务分类导航:展示服务分类(餐饮/购物/娱乐/美容/家政等),点击切换分类
- 右侧商家列表:展示当前分类下的商家列表,每个商家卡片包含商家名称、图片、服务信息
- 上拉加载更多:加载更多商家

#### 3.2.4 行囊/购物车(cart)

**核心功能**:购物车商品管理与结算

**功能模块**:
- 空状态:购物车无商品时展示空状态提示
- 按门店分组商品列表:商品按所属门店分组展示
- 门店全选:勾选后选中该门店下所有商品
- 商品复选框:单独选择商品
- 数量加减:调整商品购买数量
- 删除:删除选中商品
- 底部全选:勾选后选中所有商品
- 去结算:点击进入支付页面

#### 3.2.5 侠客我的(user)

**核心功能**:用户信息管理与功能入口

**功能模块**:
- 用户信息卡:展示武侠风昵称(随机生成或用户编辑)、会员段位(江湖散修/外门弟子/掌门等)
- 资产行:展示积分、余额、优惠券数量
- 订单统计:展示待付款/待发货/待收货/待评价/售后订单数量,点击进入订单中心对应Tab
- 侠客中心:包含我的段位、我的佣金、分销团队、邀请有礼功能入口
- 商家申请入口:展示未申请/审核中/已通过三种状态,点击进入商家入驻申请页
- 推广入口:进入推广功能页面
- 功能菜单组:包含侠客令、珍宝库、江湖事功能入口
- 武林盟入口:当用户profile.role为admin时显示,点击进入武林盟主页

## 4. 业务规则与逻辑

### 4.1 情绪匹配规则
- 用户输入情绪或选择快捷情绪词后,系统根据情绪标签匹配对应商品和内容
- 状态选择栏(时间状态/日期状态)作为辅助筛选条件
- 双列瀑布流Feed优先展示情绪匹配度高的商品,同时考虑附近优先原则

### 4.2 购物车规则
- 商品按所属门店分组展示
- 门店全选勾选后,该门店下所有商品自动选中
- 底部全选勾选后,所有门店下所有商品自动选中
- 去结算时仅结算已选中的商品

### 4.3 支付规则
- 支付页面展示订单号和支付倒计时
- 用户可选择微信支付或金豆支付
- 确认支付后完成订单支付
- 取消支付后订单保持待付款状态

### 4.4 会员段位规则
- 用户初始段位为江湖散修
- 根据用户消费金额、邀请人数等条件提升段位(外门弟子/掌门等)

### 4.5 分销规则
- 用户通过邀请有礼功能邀请新用户注册
- 新用户消费后,邀请者获得佣金
- 佣金记录在我的佣金中

### 4.6 商家入驻规则
- 商家提交入驻申请后,状态变更为审核中
- 平台审核通过后,状态变更为已通过,商家可开展业务

### 4.7 管理后台权限规则
- 所有管理后台页面需校验用户profile.role是否为admin
- 非admin用户访问管理后台页面时,跳转至首页并提示「无权限」
- 仅当用户profile.role为admin时,侠客我的页面显示武林盟入口

### 4.8 审核流程规则
- 商家申请审核:通过后商家状态变更为已通过,驳回后需填写驳回理由
- 商品审核:批准上架后商品状态变更为已上架,驳回后需填写驳回理由
- 提现审核:通过后执行代付操作,驳回后提现申请状态变更为已驳回
- UGC内容审核:审核通过后内容正常展示,审核不通过则不展示

## 5. 异常与边界情况

| 场景 | 处理方式 |
|------|----------|
| 用户未登录访问需登录页面 | 跳转至登录页 |
| 购物车为空时点击去结算 | 提示购物车为空 |
| 支付倒计时结束 | 订单自动取消,提示用户重新下单 |
| 商品库存不足 | 提示库存不足,无法加入购物车或购买 |
| 网络异常 | 提示网络异常,请稍后重试 |
| 搜索无结果 | 展示无结果提示 |
| 商家入驻申请审核失败 | 展示审核失败原因,允许重新提交 |
| 非admin用户访问管理后台 | 跳转至首页并提示「无权限」 |
| 审核操作失败 | 提示操作失败,请稍后重试 |

## 6. 验收标准

1. 用户打开小程序,进入首页,输入情绪或选择快捷情绪词,查看匹配的商品Feed
2. 用户点击商品卡片,进入商品详情页,点击加入购物车
3. 用户点击底部TabBar进入购物车页面,勾选商品,点击去结算
4. 用户进入支付页面,选择支付方式,点击确认支付,完成订单支付
5. 用户返回首页,点击侠客我的Tab,查看订单统计,点击待收货进入订单中心查看订单详情
6. admin用户登录后,在侠客我的页面点击武林盟入口,进入武林盟主页,查看待审核数量
7. admin用户点击待审商家申请卡片,进入门派大典页面,对商家申请进行通过或驳回操作
8. admin用户返回武林盟主页,点击待审商品卡片,进入宝贝审阅页面,对商品进行批准上架或驳回操作
