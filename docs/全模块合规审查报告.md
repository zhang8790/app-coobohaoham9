### 2.2 必须整改项（中度行政风险）

**【整改 1】缺失《分销服务协议》法定要件（框架第三步（二））**
当前分销规则散落于 `commission-rules` 规则页，不具备协议法律地位，且缺四要件：
- 主体界定：推广者 = 普通消费者自愿分享，**非代理商/加盟商**，平台不收加盟费/保证金；
- 分润法定表述：仅一级直接推荐 + 二级间接推荐结算，**三级及以外零佣金**，不计注册/拉新；
- 禁止行为条款：禁止以"投资/保本/升值"宣传，禁止虚构收益；
- 法律风险告知条款：明确多层级返利属传销违法，用户私设层级自行担责。

**【整改 2】"最终解释权归平台所有"格式条款（框架第三步（一））**
- `commission-rules/index.tsx:49`：`本规则最终解释权归来电有喜平台所有。`
- `rank-rules/index.tsx:65`：同款表述。
- 定性：典型不公平格式条款，市场监管可认定无效并处罚（合同违法行为监督处理办法）。**必须删除或改为"本规则解释权在法律法规框架内由平台依法行使，与用户有争议以协商/司法为准"**。

**【整改 3】用户协议单方修改 + 免责过宽（框架第三步（一）（四））**
- `user-agreement/index.tsx:64`：`我们有权根据需要修改本协议……继续使用即视为接受`——单方修改条款，建议改为"重大变更将另行显著通知并重新征得同意"。
- `user-agreement/index.tsx:57-59`：免责声明过宽（"因第三方原因不承担直接赔偿责任""不保证平台无漏洞"），建议收窄至法定免责边界（不可抗力、用户自身过错等）。

**【整改 4】隐私政策单方条款（框架 PIPL）**
- `privacy-policy/index.tsx:65`：`来电有喜保留随时修改本政策的权利`——建议改为"重大变更将重新征得您的授权同意"。
