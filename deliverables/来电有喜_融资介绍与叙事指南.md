# 来电有喜 · 融资介绍与叙事指南

> 适用场景：资本初次接触、路演、BP 投递、深度尽调
> 撰写视角：项目技术/产品主导方（最懂机制的人来讲）
> 一句话定调：**我们不是在做一个本地生活小程序，而是在数字化"人与消费之间的情绪连接"——这是最后一片还没被规模化的消费决策维度。**

---

## 一、先立人设：你该怎么开口介绍自己

资本先投人，再投事。开口 30 秒要完成三件事：**你是谁、你做的是什么品类、你为什么不一样**。

建议话术（自然版，不要背稿感）：

> "我是来电有喜的产品技术负责人。我们做的是**本地生活的情绪经济平台**——用微信小程序承载。
> 和美团、点评、抖音团购最大的不同是：它们帮人'找更便宜的货'，我们帮人'被此刻的情绪照顾到'。
> 背后是一套我们把'情绪价值'这件事**第一次做成了可标准化、可量化生产**的引擎。"

⚠️ 关键提示：
- **不要**一上来就讲技术栈、功能列表、页面数量——资本会睡着。
- **要**先抛"差异化的心智锚点"：货 vs 人、便宜 vs 被照顾、找 vs 被懂。
- 武侠视觉与"侠客/掌门"等人格化体系是**记忆点**，放到后面"品牌资产"环节讲，不要在开场堆砌。

## 二、电梯演讲（Elevator Pitch）· 三档

**10 秒版（电梯里、酒会偶遇）**
> "来电有喜，本地生活的情绪经济平台——把'我此刻需要被照顾'变成可搜索、可推荐、可确权的消费。"

**30 秒版（正式破冰）**
> "中国本地生活已经卷成了'谁更便宜'。但消费者真正的决策起点不是价格，是情绪——累了、孤单了、想犒劳自己。
> 来电有喜做的是：让用户**用一句话说出情绪**，我们识别它、匹配场景、推荐就近好物；让商家**用一套编译工具**把商品'翻译'成情绪语言；让每一次消费都**变成一次情绪确权**，用户积累成长值、平台增长反哺用户。
> 我们数字化了'货与人之间的情绪连接'。"

**1 分钟版（路演前奏/投资人要听更多）**
> "本地生活赛道被'流量分发 + 价格战'困住了。我们认为下一个增量不在'更便宜'，而在'更懂人'。
> 来电有喜用三件事构建壁垒：
> 第一，**情绪编译引擎**——把商家'会写情绪文案'的天赋，变成五维标签 + 一键生成 + 100 分质量评分的标准化生产工具，供给侧第一次可规模化；
> 第二，**情绪导购**——用户不再关键词搜索，而是'表达感受'，系统识别 6 大情绪态并推荐场景与就近好物；
> 第三，**消费即确权**——支付后用户为这次消费'确权'情绪，获得情绪豆和会员贡献值，平台增长按贡献反哺用户。
> 配套二级分销段位体系和金豆/情绪豆双货币，形成自增长飞轮。目前产品已迭代到 V5，B 端、C 端、治理端完整跑通。"

## 三、项目全景：我们解决的真问题

**不要用"痛点清单"开场（太老套），用"一个被忽视的事实"开场：**

> "所有本地生活平台都在数字化'货'和'交易'，没有任何一家在认真数字化'人和货之间的情绪'。
> 但情绪才是消费的第一性原理——人先有情绪，才有消费。"

**我们要解决的三个真问题：**

| 问题 | 现状 | 我们的解法 |
|---|---|---|
| 用户"不知道自己想要什么" | 搜索框要求用户已经想清楚（关键词） | 情绪导购：说感受 → 系统给场景 |
| 商家"不会卖情绪价值" | 情绪文案靠天赋，无法规模化、无法质控 | 情绪编译引擎：五维打标→一键编译→质量评分 |
| 平台"留不住人、靠补贴" | 用户是过客，无身份、无资产、无共建感 | 消费即确权：用户变"共建者"，贡献值反哺 |

**愿景一句话：** 让每一次本地消费，都成为被接住的情绪片刻。

## 四、产品力：三大数字化引擎（核心故事）

这是整场介绍的重头戏。用"引擎"而非"功能"来组织——资本理解引擎，不理解功能。

### 4.1 供给侧引擎 · 情绪编译工作台（壁垒核心）

**讲法：** "情绪价值营销，过去是玄学。我们把它变成了流水线。"

- 商家给商品打 **五维标签**（功能 / 场景 / 情绪 / 身份 / 感官），每维智能推荐；
- 点 **一键编译**，三阶段翻译引擎生成情绪化叙事：
  - 第一屏·场景化问句（"加班到十点，需要一口暖的？"）
  - 第二屏·状态确认（"明明很累，又不想随便对付自己？"）
  - 第三屏·身份确认（"你是再忙也会好好照顾自己的人"）
- **实时质量评分（100 分制）** 决定商品命运：
  - 标签完整度 30 + 文案合规性 30 + 场景精准度 20 + 确权可达性 20
  - ≥阈值进入**推荐池**，否则**仅店铺展示**。

**为什么这是壁垒：** 它把"会写情绪文案"这种稀缺天赋，变成了**任何商家都能用、平台能质控、质量可量化**的生产力工具。供给侧的内容质量，从"靠运气"变成"靠系统"。

### 4.2 需求侧引擎 · 情绪导购

**讲法：** "别人让用户'找货'，我们让用户'被懂'。"

- 用户表达："今天好累""一个人好无聊""心情超好"；
- 系统识别 **6 大情绪态**（耗竭态 / 孤独态 / 表达驱动态 / 平稳态 / 怀念态 / 渴望态）；
- 返回**情绪翻译文案 + 为你推荐场景卡 + 附近 Feed 流**；
- 从"人找货"升级为"**情绪 → 场景 → 货**"。

**为什么是差异化：** 搜索是"你已经知道要什么"，情绪导购是"你不知道，但我懂你此刻需要什么"。这是需求侧的结构性差异，不是 UI 差异。

### 4.3 闭环引擎 · 消费即确权经济

**讲法：** "消费结束，关系才开始。"

- 支付成功 → 引导用户为这次消费**确权情绪**（选 1–6 个情绪标签）；
- 确权即获得：**情绪豆（奖励）+ 会员贡献值 CV + 情绪徽章**；
- 贡献值沉淀为**全平台成长占比**与**年度成长回馈（情绪豆分红）**；
- 用户从"消费者"变成"平台共建者/合伙人"。

**为什么是飞轮起点：** 确权把单次交易变成**长期关系资产**。用户为了积累贡献值会复购、会分享（分享带邀请码锁客）、会拉新——留存与裂变内生于产品机制，而非靠外部补贴。

## 五、增长飞轮：货币体系 + 分销裂变

**这一段讲"单位经济"和"自增长"，资本最关心。**

### 双货币模型（清晰、合规）
- **金豆（gold_beans）**：1:1 锚定人民币，真源货币，用于消费与提现——**资产清晰、合规可控**；
- **情绪豆**：确权与成长奖励，用于会员回馈与分红——**激励层货币，不涉资金池风险**。

### 段位制二级分销（武侠人格化）
- 段位：江湖散修 → 外门弟子 → 内门弟子 → 核心弟子 → 长老 → 掌门；
- 佣金：L1 15%–28%、L2 6%–16%，**段位越高比例越大，升段永久生效**；
- 锁客：推广码 / 二维码，**分享文章也可锁定下线**，链条永久绑定；
- 风控完备：活跃门槛、退款回扣、连续未拓新衰减、严禁刷单——**不是资金盘，是带风控的合规分销**。

**飞轮逻辑一句话：** 情绪共鸣带来自然分享 → 分享锁客降低获客成本 → 确权沉淀贡献值提升 LTV → 段位成长放大分销意愿 → 更多内容被编译进推荐池 → 体验更好 → 更多情绪共鸣。

## 六、数字化思维：怎么用"数字化"框架讲这个项目

资本爱听"数字化"，但大多数人只把数字化当成"上个系统"。你要讲出**数字化思维的本质——把模糊的主观体验，变成可度量、可规模化的数据资产**。

**核心命题：**
> "情绪是消费的第一性原理，却长期是最后一片未被数字化的维度。我们做的，就是把'情绪价值'这件主观、模糊、依赖天赋的事，拆解、标签化、引擎化、确权化。"

**数字化拆解框架（讲的时候一条条点）：**

| 主观模糊的东西 | 我们的数字化动作 | 产出（数据资产） |
|---|---|---|
| "这商品很有氛围感" | 五维标签打标 | 结构化标签库 |
| "文案写得好有情绪" | 三阶段编译引擎 | 标准化情绪叙事 |
| "这条内容质量不错" | 100 分质量评分 | 可质控的推荐池准入 |
| "我此刻不知道想要啥" | 6 态情绪识别 | 情绪→场景→货的匹配 |
| "我对这单挺满意" | 消费即确权 | 情绪豆 + 贡献值 CV |
| "平台跟我有关" | 成长占比 + 分红估算 | 用户共建关系量化 |

**金句收尾：** "别人在数字化'货'和'交易'，我们在数字化'人和货之间的情绪连接'。前者是存量优化，后者是新大陆。"

## 七、差异化思维：怎么用"差异化"框架讲，对比竞品

**不要贬低竞品，要做"维度对比"——显得你格局大、想得清。**

**对比表（路演可放一页）：**

| 维度 | 传统本地生活（美团/点评/抖音团购） | 来电有喜 |
|---|---|---|
| 心智锚点 | 去哪吃/玩**更便宜** | 此刻我**需要被照顾** |
| 中心 | 货（SKU、价格） | 人（情绪、身份） |
| 发现方式 | 关键词搜索 / 低价推荐 | 情绪表达 → 场景推荐 |
| 供给侧工具 | 上架、标价、凑单 | 情绪编译引擎（质控生产） |
| 用户身份 | 过客/价格敏感者 | 侠客（段位、贡献、共建） |
| 留存逻辑 | 补贴、红包、会员折扣 | 确权资产、成长反哺 |
| 增长逻辑 | 买量、地推、补贴战 | 情绪共鸣自然裂变 |

**差异化的三层护城河（由浅入深讲）：**
1. **体验差异**（表层）：表达情绪 vs 搜关键词——感知即不同；
2. **生产差异**（中层）：情绪编译引擎是竞品没有的**供给侧生产力工具**，构成内容质量壁垒；
3. **关系差异**（深层）：消费即确权让用户从"被收割的流量"变成"共建分成的合伙人"——这是**结构性关系重构**，极难抄袭。

**金句收尾：** "它们优化的是'交易效率'，我们重构的是'消费关系'。效率可以被追赶，关系一旦建立就是护城河。"

## 八、正式路演叙事脚本（故事线 · 约 8–10 分钟）

**第一幕 · 钩子（1 分钟）**
讲"被忽视的事实"：所有平台数字化了货和交易，没人数字化情绪。情绪才是消费第一性原理。

**第二幕 · 用户故事（2 分钟）**
带一个真实感的人物："加班的林小姐，累了但说不清要什么"→ 她打开来电有喜，说"好累"→ 系统识别耗竭态 → 推了一家就近的暖汤馆 → 喝完，为这次消费确权"被治愈" → 拿到情绪豆和贡献值。
*（用情绪导购 + 确权讲完 C 端价值）*

**第三幕 · 商家故事（2 分钟）**
"那家暖汤馆的老板不会写文案"→ 用情绪编译工作台，五维打标、一键生成三屏情绪叙事 → 实时评分进推荐池 → 门店客流提升。
*（讲清供给侧引擎与壁垒）*

**第四幕 · 商业模式（2 分钟）**
双货币、段位分销、确权反哺 → 自增长飞轮 → 单位经济健康、风控合规。
*（讲清怎么赚钱、怎么增长、为什么可持续）*

**第五幕 · 为什么是现在 + 要什么（1–2 分钟）**
情绪经济是确定性趋势；我们已 V5 跑通 B/C/治理三端；本轮用于城市复制、引擎算法深化、品牌势能。明确要"多少钱、换什么、达成什么里程碑"。

## 十、投资人常见 Q&A 与应对

**Q1：这不就是又一个团购/内容电商？**
A：不是。团购优化"交易效率"，我们重构"消费关系"。我们的用户是"侠客"、有贡献值、享平台成长反哺——这是结构性关系差异，不是又一家卖券的。

**Q2：情绪识别准吗？会不会很鸡肋？**
A：我们用"6 大情绪态 + 30 个标准标签"做轻量可解释的匹配，不依赖重模型、可冷启动、可质控。它服务于"推荐相关性"而非"心理诊断"，落地稳健。

**Q3：分销会不会涉传销/资金盘？**
A：我们是二级、带风控（活跃门槛、退款回扣、衰减、禁刷单）、实物消费确权触发、金豆锚定人民币合规可控。本质是合规的推广分润，不是层级返利盘。

**Q4：壁垒到底在哪？别人抄很快吧？**
A：表层体验易抄，但三层难抄——①情绪编译引擎是供给侧生产力工具（商家迁移成本+内容质量壁垒）；②确权经济把用户变成共建者（关系重构）；③武侠人格化品牌资产（认知壁垒）。三者叠加形成网络效应。

**Q5：单位经济模型健康吗？**
A：双货币隔离了激励层与资金层；分销佣金由真实成交触发、售后周期结算；确权奖励发的是情绪豆（成长层），不侵蚀本金。模型可随规模摊薄获客成本（情绪裂变自然获客）。

**Q6：技术团队和可持续性？**
A：Taro 跨端（小程序+H5）、Supabase 数据底座、已迭代 V5，工程体系完整可规模复制。本轮重点是把"单城模型"复制为多城网络。

## 附：讲述时的三个"不要"

1. **不要先堆功能列表**——用引擎和飞轮组织，功能是注脚；
2. **不要贬低竞品**——做维度对比，显格局；
3. **不要回避合规**——主动讲双货币隔离、风控、确权触发，资本最怕隐形雷。

> 记住你的核心叙事锚点：**"别人数字化货与交易，我们数字化人与货之间的情绪。"** 这句话，全场反复回扣，它就成为了投资人对你的记忆点。
