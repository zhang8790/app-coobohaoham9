### 2.2 小程序（暖奶油国潮）— `src/app.scss` `:root`（HSL 空格语法）

核心：`--background: 33 100% 99%`（奶油）、`--foreground: 20 9% 11%`（近黑）、
`--primary: 17 87% 41%`（= `#C2410C`）、`--muted`/`--border`/`--destructive` 等。

**使用方式**：
- Tailwind class：`className="bg-background text-foreground border-border text-primary"`
- 内联 style：`style={{ color: 'hsl(var(--primary))' }}`

### 2.3 品牌色对齐结论

- 赤陶橙 `#C2410C` = `hsl(17 87% 41%)` = 两端 `--primary` → **视觉完全一致**
- 微信绿 `#07C160`、金豆琥珀 `#D97706` 为**产品语义固定色**，保留原值（不强行 token 化）

## 三、图标规范（AI slop 红线）

| 规则 | 状态 |
|---|---|
| ❌ 禁止 emoji 当 UI 图标（导航/按钮/状态徽章）| 强制 |
| ✅ 后台用内联 SVG（feather 风格）| 已实现 `components/icons.tsx` 的 `NavIcon` 组件，stroke=`currentColor` 自动继承 active 色 |
| ✅ 小程序用 iconify class（`i-mdi-*`）或 SVG | 既有规范 |
| ✅ 共享图标映射 | grid/store/box/dollar/news/user/refund/megaphone/shield/chart/users/document/book/bank/trending/tea/chat/building/calculator/check/alert/x |

**新增图标流程**：在 `icons.tsx` 的 `ICON_PATHS` 加 feather path → 页面用 `<NavIcon name="xxx" />`。

## 七、后续开发约定

1. 后台新页面：颜色**只用** `var(--token)`，图标**只用** `<NavIcon name="..." />`，**禁止新增 emoji 图标**。
2. 小程序新页面：用 Tailwind 类（`bg-background`/`text-foreground`）或 `hsl(var(--token))`，禁止裸 `#`。
3. 品牌名统一用「来电有喜」，后台副标题用「管理后台」。
4. 任何 UI 改动先对照本规范，确保三端调性一致。
